# Admin Guide

The admin panel is private and intended for one owner.

Local URL:

```text
http://localhost:3000/admin/login
```

Production URL:

```text
https://your-domain.com/admin/login
```

## Sections

`Profile`

Your name, role, short bio, about text, location, email, and focus statement.

`Hero`

The first impression on the homepage: eyebrow, headline, subheadline, buttons, and AI/ML chips.

`Skills`

Skill categories and individual skills. Add a new skill here whenever you learn something new.

`Projects`

Project cards for the public site. Each project can include GitHub, live demo, and external case-study links. Projects can be visible, featured, or hidden.

`Demos`

Placeholder for future demos. Keep demos hidden until you have real demos.

`Experience`, `Education`, `Certifications`

Career and academic sections for professional credibility.

`Achievements`

Awards, hackathons, competitions, scholarships, and recognition. Entries are hidden publicly unless marked visible.

`Resume`

Upload the direct PDF resume and edit resume highlights. The public `/resume` page previews and downloads this file.

`Social Links`

GitHub, LinkedIn, email, or other public links.

`Contact Inbox`

Review messages from the public contact form. Mark messages read/unread or archive them. Archived messages are kept for reference.

`Settings`

Site name, SEO title, SEO description, and nav settings.

## Recommended Editing Workflow

1. Update one section at a time.
2. Save in the admin panel.
3. Open the public page and review it.
4. Keep unfinished projects, demos, achievements, and hackathons hidden.
5. Only publish visible content that supports your AI/ML engineering identity.

## Content Rules For A Strong Portfolio

- Lead with AI Engineering, ML Engineering, LLMs, and MLOps.
- Prefer project outcomes over generic descriptions.
- Add case-study links only when the external page is ready.
- Keep project cards concise.
- Upload a current resume before sharing the site with employers.
- Avoid empty demo links; hidden is better than unfinished.
